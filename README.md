## v1.6（2026-06-11）— 弹幕控制模块 · WebSocket 直播弹幕操控

### 新功能：B站直播弹幕控制模块

- **B站直播间弹幕实时监听**：输入直播间ID即可连接弹幕服务器，实时接收并显示弹幕
- **弹幕指令触发键鼠动作**：配置弹幕关键词 → 自动执行键盘/鼠标操作（按键、组合键、长按、鼠标点击、移动、滚轮）
- **内置默认指令**：左键/右键/E技能/Ctrl技能，即装即用
- **指令管理面板**：可视化添加/编辑/删除/启用指令，支持精确匹配、前缀匹配、正则匹配
- **权限与频率控制**：支持仅房管/仅粉丝牌触发、全局冷却、单指令CD、粉丝牌等级限制
- **多指令匹配模式**：全部匹配或首个匹配，灵活可控

### B站 API 对接

- **WBI 签名机制**：自动获取 img_key/sub_key，对弹幕 API 请求进行 MD5 签名
- **buvid 自动获取**：未登录时自动从 B站 SPI 接口获取游客 buvid，无需 Cookie 也可连接弹幕
- **Cookie 兼容**：支持填写 Cookie 获取登录态（uid/buvid3 自动提取），游客模式也完整可用

### 弹幕连接稳定性

- **SSL 证书绕过**：PyInstaller 打包后在新电脑可能缺少系统 CA 证书，自动降级为无验证模式
- **protover 自适应**：根据 brotli 模块可用性自动选择压缩协议（brotli→protover 3，zlib→protover 2）
- **认证回包处理**：完整处理 AUTH_REPLY 和 Close 帧，精确区分"token 过期"与"认证被拒"
- **自动重连**：连接断开后自动重试，带递增间隔

### 键鼠动作执行

- **独立动作线程**：pynput 在专用线程中运行，避免主线程阻塞
- **动作队列**：异步队列解耦，弹幕触发不阻塞消息接收
- **支持 Windows 全系列**：XP/Vista/7/8/10/11 均可用

### 视觉改进

- **暗色指令列表表格**：深蓝背景 + 白色文字，选中行红色高亮
- **动作类型中文化**：键盘按键/组合键/长按/鼠标点击/鼠标移动/滚轮
- **默认房间号 21854799**（用户常用直播间）

### 安装包

- 安装后占用约 34 MB（压缩后 9.87 MB）
- 支持标准安装和最小安装

## v1.5（2026-06-04）— 部署模式升级 · 多文件安装

### 部署方式变更

- **PyInstaller 打包模式从 `onefile`（单文件）切换至 `onedir`（多文件目录）**
  - 安装后目录包含完整文件结构：`_internal/`（Python 库、DLL）、`icon.ico`、`user_manual.html` 等
  - 用户可直接在安装目录中查看 SQLite 数据库及所有依赖文件
  - 启动速度更快：无需每次运行时解压至临时目录
  - 降低杀软误报风险

### 安装包

- 安装后占用约 32 MB（压缩后 8.88 MB）
- 安装目录结构清晰，便于用户浏览和管理文件

## v1.4（2026-06-04）— 架构升级 · 数据持久化重构

### 重大变更

- **数据存储架构重构：从 JSON 文件迁移至 SQLite 数据库**
  - 新建 `DatabaseManager` 统一数据管理模块
  - 旧版 JSON 数据（`skill_pool.json`、`killer_pool.json`、`bets.json`）首次启动自动导入 SQLite，无缝升级
  - 数据目录从 `%APPDATA%\DbdOzcen\` 统一至 `%APPDATA%\黎明杀机ocean\`

### 性能与稳定性

- 启用 SQLite WAL（Write-Ahead Logging）模式，提升并发读写性能
- 添加线程锁保护，确保多线程环境下的数据安全
- 引入外键约束，杀手与条件数据级联维护
- 数据库自动建表与 schema 版本管理

### 构建与安装

- 修复 NSIS 安装脚本编码问题：脚本保存为 UTF-8 with BOM，解决中文编译报错
- 安装包大小：12.78 MB

## v1.3（2026-06-04）— 品牌重塑 · 安装体验升级

### 品牌与命名

- **正式更名为「黎明杀机ocean」**（原「黎明杀机Ozcen」）
- 统一全应用命名、安装路径、注册表项、快捷方式
- 安装程序文件名更新为 `Setup_LMOcean_v1.3.exe`

### 安装程序重构

- **安装框架从 Inno Setup 切换至 NSIS（Nullsoft Scriptable Install System）**
- 支持标准安装 / 最小安装两种安装类型
- 最小安装仅包含主程序，不安装用户手册
- 开始菜单快捷方式 + 桌面快捷方式 + 注册表卸载信息
- 卸载时可选保留用户数据，方便重装后继续使用

### 功能增强

- 新增全局快捷键系统（Ctrl+D 抽取技能 / Ctrl+K 抽取杀手 / Ctrl+B 竞猜 / Ctrl+R 重置 / Ctrl+L 解锁 / Ctrl+Q 退出）
- 新增顶部菜单栏（文件菜单 + 帮助菜单）
- 状态栏实时显示技能池/杀手/参与者数量
- 新增「关于」对话框
- 内置用户手册（HTML 格式，可选安装）
- 程序图标（icon.ico）同时用于安装程序和快捷方式

### 视觉优化

- 全新暗黑主题 UI（深蓝底色 + 红色强调色）
- 技能卡片 2×2 网格布局，视觉清晰
- 锁定/解锁图标采用程序自绘 Canvas 图形，不依赖系统字体
- 窗口最小尺寸限制（860×580），防止布局错乱

## v1.0（2026-06-04）— 初始发布

### 核心功能

- **技能随机抽取器**：从内置 126 个默认技能池中随机抽取 4 个技能，支持锁定机制
- **随机杀手抽取器**：从内置 39 个默认杀手池中随机抽取，附带随机条件
- **竞猜系统**：多人竞猜，参与者使用血点（BP）押注存活/倒地结果，押中获得 1.2 倍奖励

### 技术架构

- Python + Tkinter 构建，PyInstaller 打包为独立 exe
- 数据以 JSON 文件存储于 `%APPDATA%\DbdOzcen\`
- Inno Setup 安装程序，支持自动创建快捷方式和卸载
- 安装包大小：10.83 MB
